# 谁是卧底 - 无境探险

一款基于网页的线下聚会游戏，支持手机浏览器直接打开游玩。

## 🎮 游戏玩法

### 角色说明
- **👨‍🌾 平民**：获得多数派词语
- **🕵️ 卧底**：获得少数派词语（与平民词语相似但不同）

### 游戏流程

1. **设置玩家人数**（3-12 人）
2. **开始游戏** - 系统自动分配角色和词语
3. **传递看词** - 玩家依次传递手机，点击卡牌查看自己的词语并拍照
4. **线下发言** - 每位玩家轮流描述自己的词语
5. **投票处决** - 投票选出怀疑的卧底，点击头像后选择"处决"
6. **判断胜负**：
   - 所有卧底被淘汰 → 平民获胜
   - 所有平民被淘汰 → 卧底获胜

### 操作说明

| 操作 | 说明 |
|------|------|
| 点击卡牌 | 翻开卡牌查看词语 |
| 点击"我记住了" | 确认词语并拍照 |
| 点击玩家头像 | 选中该玩家（其他变暗） |
| 点击"忘词" | 重新查看该玩家词语（3 秒） |
| 点击"处决" | 淘汰选中玩家，显示身份 |

## 🚀 快速开始

### 本地运行
```bash
# 直接打开
open index.html

# 或使用本地服务器
python -m http.server 8000
# 访问 http://localhost:8000
```

### 手机测试（同一 WiFi）
```bash
# 查看电脑 IP
ipconfig  # Windows
ifconfig  # Mac/Linux

# 手机浏览器访问
http://<你的IP地址>:8000
```

### 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 进入 Settings → Pages
3. Source 选择 `main` 分支
4. 访问 `https://你的用户名.github.io/仓库名/`

## ⚠️ 注意事项

1. **摄像头权限**：首次使用需要授权摄像头权限用于拍照
2. **保持屏幕常亮**：游戏过程中会自动请求屏幕常亮
3. **本地存储**：游戏数据保存在本地，清除缓存会丢失
4. **离线使用**：首次加载后可断网使用（需添加 PWA 支持后完全离线）
