# 谁是卧底 - 无境探险

> **🎮 立即游玩：** [https://bushnerd.github.io/WhoIsUndercover/](https://bushnerd.github.io/WhoIsUndercover/)

一款基于网页的线下聚会游戏，支持手机浏览器直接打开游玩。

## 🎮 游戏玩法

### 📜 游戏起源

**谁是卧底**是一款流行的社交推理聚会游戏：
- **起源**：从"捉鬼游戏"（杀人游戏/Mafia）演变而来
- **命名**：由湖南卫视《快乐大本营》于**2012 年**改编并正式命名
- **国际版**：英文名 **Undercover** 或 **Who is the Spy**
- **类型**：与狼人杀、Among Us 同属"社交推理游戏"类别

### 角色说明

| 角色 | 说明 |
|------|------|
| **👨‍🌾 平民** | 获得多数派词语（人数最多），目标是找出并淘汰所有卧底和白板 |
| **🕵️ 卧底** | 获得少数派词语（与平民词语相似但不同），目标是存活到最后 |
| **🃏 白板** | 没有词语（空白牌），需要靠听别人描述来伪装，目标是存活到最后或猜对平民词语 |

### 胜利条件

| 阵营 | 获胜条件 |
|------|----------|
| **平民** | 淘汰所有卧底和白板 |
| **卧底** | 存活至场上仅剩 1 名平民 |
| **白板** | 存活到最后（可单独获胜）或在被投票时猜对平民词语（直接获胜） |
| **阵营共赢** | 若场上仅剩 1 平民 + 卧底 + 白板 → 卧底与白板共同获胜 |

### 人数配置规则

| 总人数 | 卧底（自动） | 白板（可选） | 平民（自动） |
|--------|-------------|-------------|-------------|
| 3-5 人 | 1 | 0 或 1 | 剩余 |
| 6-8 人 | 2 | 0 或 1 | 剩余 |
| 9-12 人 | 3 | 0 或 1 | 剩余 |

**计算公式：** `平民 = 总人数 - 卧底数 - 白板数`

### 游戏流程详解

1. **发词阶段** - 每位玩家抽取身份牌，查看自己的词语（卧底/白板词语不同或空白）
2. **描述阶段** - 每人每轮用**一句话或一个词**描述自己的词语（不能直接说出）
3. **讨论阶段** - 所有玩家自由讨论、分析、辩论
4. **投票阶段** - 投票选出疑似卧底/白板的玩家
5. **淘汰阶段** - 得票最多者出局，公布身份
6. **白板特殊规则** - 若淘汰的是白板，给 1 次机会猜平民词语，猜对则白板直接获胜
7. **循环** - 重复步骤 2-6 直到某一方获胜

### 词语示例

| 平民词 | 卧底词 |
|--------|--------|
| 饺子 | 包子 |
| 手机 | 电话 |
| 咖啡 | 奶茶 |
| 地铁 | 火车 |
| 微信 | QQ |

### 操作说明

| 操作 | 说明 |
|------|------|
| 点击卡牌 | 翻开卡牌查看词语 |
| 点击"我记住了" | 确认词语并拍照 |
| 点击玩家头像 | 选中该玩家（其他变暗） |
| 点击"忘词" | 重新查看该玩家词语（3 秒） |
| 点击"处决" | 淘汰选中玩家，显示身份 |

### 游戏流程（应用内）

1. **设置玩家人数**（3-12 人）
2. **开始游戏** - 系统自动分配角色和词语
3. **传递看词** - 玩家依次传递手机，点击卡牌查看自己的词语并拍照
4. **线下发言** - 每位玩家轮流描述自己的词语
5. **投票处决** - 投票选出怀疑的卧底，点击头像后选择"处决"
6. **判断胜负** - 游戏自动判定并显示结果

## 🚀 快速开始

### 本地运行
```bash
# 直接打开
open index.html

# 或使用本地服务器
python -m http.server 8000
# 访问 http://localhost:8000
```

### 手机测试（同一 WiFi）
```bash
# 查看电脑 IP
ipconfig  # Windows
ifconfig  # Mac/Linux

# 手机浏览器访问
http://<你的IP地址>:8000
```

### 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 进入 Settings → Pages
3. Source 选择 `main` 分支
4. 访问 `https://你的用户名.github.io/仓库名/`
